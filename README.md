# Homebrew Tap for Claude Code Statusline

This tap provides the Homebrew formula for [Claude Code Enhanced Statusline](https://github.com/rz1989s/claude-code-statusline).

## Installation

```bash
brew tap rz1989s/tap
brew install claude-code-statusline
```

## Usage

After installation, run the full setup:

```bash
curl -sSfL https://raw.githubusercontent.com/rz1989s/claude-code-statusline/main/install.sh | bash
```

Or verify the installation:

```bash
claude-statusline --help
```

## Updating

```bash
brew update
brew upgrade claude-code-statusline
```

## Uninstalling

```bash
brew uninstall claude-code-statusline
brew untap rz1989s/tap
```
